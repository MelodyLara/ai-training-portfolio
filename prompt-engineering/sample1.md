# Prompt Engineering Example: Medical Explanation

## Initial Prompt
Explain diabetes.

## Issues
- Too vague
- No target audience
- Leads to inconsistent outputs

## Optimized Prompt
Explain Type 2 Diabetes in simple terms for a patient. Include causes, symptoms, complications, and prevention. Avoid medical jargon.

## Result
- Structured response
- Improved clarity
- Reduced hallucinations

## Key Insight
Adding constraints and audience context significantly improves output quality.
