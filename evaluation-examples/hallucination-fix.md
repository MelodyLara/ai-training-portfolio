# AI Evaluation Example: Hallucination Detection

## Task
Evaluate accuracy of medical response.

## Model Output (Before)
"Hypertension is caused only by stress and can be cured permanently with rest."

## Issues Identified
- Factually incorrect
- Oversimplified
- Potentially unsafe

## Improved Output (After Feedback)
Hypertension is influenced by multiple factors including genetics, diet, and lifestyle. It can be managed with medication and lifestyle changes.

## Evaluation Metrics
- Accuracy: 70% → 95%
- Safety: Improved
- Completeness: Improved

## Key Insight
Targeted feedback reduces misinformation and improves model reliability.
