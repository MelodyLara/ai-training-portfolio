## 1. Prompt Engineering Samples

### Example 1: Improving Medical Explanation Accuracy

**Initial Prompt:**
Explain hypertension.

**Optimized Prompt:**
Explain hypertension in simple terms for a patient, including causes, symptoms, risks, and when to seek medical care. Avoid medical jargon.

**Impact:**

* Improved clarity and structure
* Reduced hallucinations
* Increased patient-readability

---

### Example 2: Technical Problem Solving

**Initial Prompt:**
How does a database work?

**Optimized Prompt:**
Explain how relational databases work, including tables, keys, and indexing. Provide a real-world example and keep the explanation concise.

**Impact:**

* More structured output
* Better technical depth
* Improved relevance
